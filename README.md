# 📖 About

**BMW Experience** is a modern landing page inspired by BMW's premium design language.

The project was developed to explore advanced front-end animation techniques using **GSAP**, creating an engaging experience through smooth transitions, scroll effects and cinematic interactions.

This project focuses on combining visual storytelling with responsive design using only HTML, CSS and Vanilla JavaScript.
